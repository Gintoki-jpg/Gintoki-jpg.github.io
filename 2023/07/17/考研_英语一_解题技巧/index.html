考研英语主要有以下五大部分题型：

  • 完形填空
  • 阅读理解
  • 新题型
  • 翻译英译汉
  • 写作，共2题，分为大小作文，大作文20分（图画类），小作文10分（书信通知类）

一般按照顺序做题是做不完的，推荐按照以下流程解题：

  1. 小作文：15min
  2. 大作文：35min
  3. 阅读理解：60min
  4. 新题型：10min
  5. 翻译20min
  6. 完形填空20min

下面分别介绍每个部分题型对应的技巧（再次声明不要过度依赖技巧，阅读能力才是根本之道）

1.完形填空

首先是做完形的原则：

  • 从语法角度分析句子，确定空格成分；从文意角度分析句子，推测空格的含义，从选项中找到最佳答案

  • 完型填空分值低难度大，通常放在最后做，时间控制在10~15min

  • 精读前1~2句话，必定是主旨句，只有确定了主旨之后的空才能做（这和阅读理解主旨句不定不同）

  • 以段为单位做题，题目选做优先选容易的，后续结合技巧查漏补缺

完形填空总结了如下技巧

  1. “棒打鸳鸯”即物理上的拆散搭配，其解决方法不难，就是去掉无聊的修饰性成分等找主干；
  2. 单句单题往往来说比较简单，大部分情况下这种题的答案与前后句或全文中心思想关系不大；
  3. 针对长难句（即一个句子中设置多个选项），可以从后往前做，逐个击破；
  4. 考研完型填空文章通常”完美”，通篇作者态度一致，从一个高度去指导整个行文。有些题目没有具体线索可以找到时，可以从作者态度去判断答案。在最近几年比较多见，尤其是最开头的题目，遇到也不要害怕；
  5. 绝对同义选项两个都错；相对同义选项，也就是约等于选项，这就是出题点或是题眼，细微的差别就是考试的考点，二选一其中一个必须对；
  6. 如出现陌生的选项词汇或四个选项都相近的，选择相对简单的词汇，不要选择偏的怪的词汇；

然后是一些比较“虚幻”的技巧：

  1. 一篇完形填空答案ABCD 的数量通常各有4-6个，5个更常见。
    例如：如果还剩2题不会做，C这个选项已经选了6个了，D这个选项已经选了5个了，那剩下的两道题就尽量不要蒙C或D，蒙A或者B的正确概率更高
  2. 一篇完形填空连续三道题都选一样的（都选A，都选B，都选C，都选D）可能性很小，连续两道题都选一样的可能会有0-3处。
    例如：如果你发现你已选的答案连续三个或者四个都是一样的，那你这些答案里面选错了的可能性比较大，要重新再看一下题。
  3. 一般A作为正确选项的个数最多，基本没有连续三个答案都一样的情况。
  4. 完形填空五个题为一个题组，一个题组中出现三个相同答案的概率也很低。

2.阅读理解

首先是做阅读的方式，规定一天完成一篇阅读即可，一套完整的流程如下：

  1. 第一遍做真题的时候计时；
  2. 真题做完之后查看答案的单词，将整篇文章中搞不懂的单词的意思全部查阅出来；
  3. 将单词的意思全部搞懂之后再次做一遍选项，相当于第二遍做题，做完之后对答案，给出每个选项为什么不正确；
  4. 全文翻译，逐词逐句进行，对于长难句需要理清楚其语法逻辑关系；
  5. 复盘整理，将前面查阅过的单词单独再次整理在每日的单词本上，每天睡觉之前复习一遍；

然后是解题步骤，先前一直是拿到文章之后直接开始通读前几段，读不下去了然后开始看选项，然后做了前面的选项之后再继续看之后的文章解之后的选项。这种方式其实比较适合之前四六级中的为对应文章选择对应的段落大意那种题型，针对一般的阅读理解来说，推荐按照以下解题步骤进行：

  1. 通读题干（不要看选项！只看题干）：通过题干可以把握整篇文章讲的是什么，同时还能够抓住阅读重点，对于一些不重要的段落信息可以直接跳过减轻阅读负担提高阅读速度；
  2. 通读全文，掌握文章的主旨和每个段落的大意（注意仔细阅读各个段落的首句）
    • 对于短时记忆较差的，可以在阅读过程中暂停阅读并快速对之前的已读信息进行总结（这个过程要快，不要逐字逐句的在那里慢慢总结）
    • 对于文章的细节过关键词（一般是长难句的主语）可以适当做标记
  3. 接下来就是解题过程，通过题干中的关键词可以定位到文章中答案的位置，仔细阅读其上下文确定答案
    • 一定要将题干（注意不是选项）代回到原文找到其出处（换句话说需要找到能够证明所选选项一定正确的证据），不能凭借经验或印象解题，更不能直接猜答案（主观臆断）

阅读理解主要如下题型，下面分别介绍其对应的解题方法、解题步骤和解题要点

  1. 中心主旨题：一般在问文章的中心思想是什么、文章在讨论什么或取什么样的标题最好
    • 主旨题的正确选项与文章主旨是全面匹配的。换句话说，正确答案里应该包含所有的中心词/句，而错误选项/干扰项往往只包含部分，信息是片面且不完整的；
    • 不要觉得某个选项里的关键词好像跟主旨词长得像就选它，有可能是出题人故意用形近词埋下的坑；
    • 另一类主旨题是针对某一部分的主旨，比如问作者举这个例子的目的是什么，一定不要去选与这个例子相关的选项如“大脑情感区在梦形成时的作用”“大脑情感区与非情感区的区别”等，应当选择脱离于这些例子（即离开了这些例子仍成立或可以使用其他例子同样表达的观点）的选项如“情感与梦境的关系”
  2. 态度观点题：主要考查“理解作者意图、观点或态度”的能力，而作者态度或观点有3种：赞成、反对或中立
    • 像uninterested、unconcerned、indifferent等表示“不感兴趣”或confused、puzzled等表示“困惑疑惑”，或biased、prejudice、contemptuous等表达态度较为偏激的一般都是错误选项
  3. 推理判断题：主要考查判断和推理的能力
    • 虽然叫推理题，但并不是让你用自己的理解对文章进行推理，很多时候答案就是原文内容，只不过改写了某些细节信息而已
    • 选项中出现很绝对的词，比如must、never等要警惕，大概率是正确选项的用词都比较委婉，如may、might等
    • 重视题目对应段落或全文中重复出现的信息，重复的是重点，重点必然是考点
    • 文中的转折处是命题热点
  4. 细节题：细节题主要考查的是能否理解文中具体信息的能力，题目占比最高且出题方式很灵活，相比于词汇题或例证题来说题干信息也不太好定位，所以做这类题时要尤其谨慎
    • 定位：根据题干信息找到文中对应的话或段落。注意只有当原文中的信息和题干全部一一对应的时候才算定位准确
    • 对比：将原文与4个选项进行对比，选出正确答案。选项一般会对原文进行改写或替换，比如正话反说、同/近义替换、上下义词替换等
  5. 词汇题：词汇题一般要求我们猜测划线单词/短语的含义，这类题的核心解题思路是联系上下文，看文中划线单词的上一句和下一句话，判断前后逻辑关系做题
    • 如果前后逻辑一致，如因果关系、并列关系、顺承关系等就在前后寻找考察词汇的同义词/近义词；
    • 如果前后句子逻辑关系是相反的，比如对比关系、转折关系、让步关系等，要在前后寻找它的反义词；
    • 注意转折和让步的区别，转折词后面的句子的意思彻底推翻了转折词前的句子，而让步后面句子的意思不能推翻让步前面那句话的意思

最后是一些“野路子”技巧（适用于有时候觉得几个选项都对不知如何是好 – 前提是读懂文章）：

  1. 阅读理解并不是让你代入自己的理解，而是阅读之后，从作者的角度去理解整篇文章，所以不要带有自己的常识以及情感去解题
  2. 文章基本上按顺序出题，一般来说一段一个题（这是常识，解题顺序依次就行）
  3. 原句出现或原词复现基本都不是正确选项（即原词复现应当关注具体复现的内容 – 比高考英语更上一层楼）
  4. 答案分布规律（这都是记着玩的，别太当真）
    • 相同答案不相邻规律：两个相邻题目答案相同概率很小，一般二十道题里多出现一到两种这样的情况，至于三个相同答案相邻的情况更是罕见
    • 选项均匀分布规律：每一篇答案分布中，五个中至少出现ABCD中的三个，且五个答案至少有两个相同
    • 接口答案不同规律：每一篇的最后一题与下一篇的第一题答案不同

3.新题型

新题型又称段落大意题，新题型的解答需要理解文章的段落大意。先速读文章，先弄清文章主旨和大体意思;然后去看读被插入的句子，找出句首或句尾的关键词，只有知道句子的大概意思才能正确的匹配文章;紧接着阅读文章，并了解空格前后的信息，寻找与插入句相关联的意思;最后经过反复甄选之后选择与插入句最相关的空格，确保文章意思连贯。

3.1 七选五

七选五主要考查文章逻辑的一致性，一般拿走的空白处与上下文之间都有明显的逻辑关系或内容联系。关键信息指的是文中重要的句子和词汇，比如说核心名词、动词、形容词以及一些地名、数字等。可以通过代词、名词和逻辑关系词(包括并列、转折、因果、递进等)检查上下文的逻辑。

3.2 小标题对应

小标题有可能是对一段话大意的概括，也可能是某个观点的再现，做题时主要看标题与段落的一致性。

3.3 排序题

排序题考查对文章整体结构的把握，做题的时候要着重理解选项和上下文之间在内容和逻辑上的联系，用连贯性解题。注意排序题不需要通读全文，这样不仅会浪费时间，还会扰乱你的关键信息，适得其反。另外，排序题基本都会给出两个已知序号，这两个已知的序号是解题的关键，同时要注意识记重要的逻辑词。

4.翻译

5.写作

关于时态：不管是大作文还是小作文，时态以一般现在时为主。图表描述（过去年代的数据）和举例论证（曾经发生的事）会用到过去时。文章结尾展望未来可使用将来时。

5.1 应用文

5.1.1 书信类

书信的段落通常较短，首段和末段一般只包含一两句话。开头的段落用一句话说明事由，必要时，可以再加一句话提及以前的书信联系。回复书信时，最好说明何时收到关于何事的书信。

考研大纲明确规定，考研写作考试只需从称呼写起，并根据题目要求写出正文、结束语，签名一般统一用“Li Ming”（李明）这个名字。一般的书信格式如下：

  1. 称呼：称呼是书信的开头部分，一般用Dear…开头，注意头衔的使用。考研写作要求称呼后用逗号，不能用冒号。
  2. 正文：正文是书信中最重要的内容，也是应用文的得分关键。考研应用文一般会交代写作背景，给出所应包含的内容要点。因此，考生应严格按照提纲要点来完成写作，顺序不要打乱。
  3. 结束语：结束语是表示礼节的套语，第一个字母要大写，后面要加逗号。
  4. 签名：千万不要写出自己的真实姓名，且考研英语不需要写日期。

5.1.2 告示类

告示主要分为以下四大部分：

  1. 标题：书信类应用文不需要写标题，但告示类应用文必须写标题，这点非常重要。需要说明的是，告示的标题一般为短语。其中实词（有实际意义的词，如名词、动词、形容词等）第一个字母需要大写，虚词（没有实际意义的词，如介词、冠词、连词等）除作为短语首词外均不需大写。
  2. 书信类应用文不要求写日期，但告示类应用文一般需要注明日期，写在标题下一行的右侧，右端与右边线对齐。
  3. 正文：根据题目要求，写一至三段。
  4. 署名：书信类应用文签名一般是个人，但告示类应用文的署名可以是个人，也可以是公告单位。后者的署名中实词首字母需大写，虚词除作为短语首词外均不需要大写。

5.1.3 纪要类

纪要的写作格式一般包括标题、会议时间、会议地点、出席人、主持人、会议概要、提交人和提交日期。如果考到纪要，格式应该不是考查重点，一般不会扣分，只写标题、正文、落款也可，不用太过担心。

5.2 大作文

大作文主要有图画作文、图表作文和材料作文三种，除了第一段写法略有不同外，其余段均类似（图画作文第一段一般是仅客观描述图表，图表作文第一段一般是仅客观描述图表，材料作文第一段分别进行背景交代、引出主题和确定论点）

5.2.1 图画类

图画类大作文分为三段式进行写作，基本框架结构如下：

  1. Introduction：图画作文第一段一般是图画描述，不应发表太多议论，第一段发表议论属于跑题。
    • 总体描述：大致描述图像内容
    • 挖掘细节词：进一步描述图画细节
    • 文字说明：大部分图画都配有中文的文字说明，可翻译作为首段末句
  2. Body：图画作文的创作原则是：以小见大，把握象征寓意，个体想为群体，具体想为抽象。
    • 主题句：揭露图画的深层象征寓意
    • 论证：切忌只有论点没有论据，每个观点后都应当有原因、解释或例证
      • 五大角度：背景（政治、经济、文化和社会背景）、事实（普遍公认的社会现象）、反方观点（考生作为正方，可以提出反方观点进行驳斥）、原因、结果
      • 六种手段：因果论证、分类论证、定义和解释（自问自答或换个角度解释论点）、统计和事实（可以适当虚构数据）、举例论证、比较和对比（比较两者相同点、对比两者不同点）
      • 联系现实：将图中的每个事物与现实进行联系
    • 小结：对论证进行总结，与主题句呼应
  3. Conclusion：
    • 结论句
    • 建议或评论
    • 包装结尾，展望未来：可以使用倒装句、强调句或虚拟语气

5.2.2 图表类

（图表类作为了解即可，是英二的重点。）

考生应该仔细观察、分析图表所给出的大量信息，从中选取最重要、最有代表性、最核心的信息，然后根据全文的主旨去组织运用所获取的关键信息。考生应切忌简单地罗列图表所给出的信息。以下写作要点更应注意：

  • 图表题型写作一般采用一般现在时，但如果图表给出了具体时间参照，考生则应对时态进行相应的调整。

  • 图表题型可以细分为表格图、曲线图、柱状图和饼状图。除了上述共同要点，考生还应弄懂这四种图各自在写作方面的特点

    • 表格图题型要求考生对表格中所给出的大量数字进行比较分析，从中找出其变化规律。
      • 首尾呼应法。只看第一点和最后一点数据，只点首尾，说清即可。中间的非典型性数据不用提及。
    • 曲线图题型要求考生认真观察坐标系所显示的数据信息，并且密切注意交汇在坐标横轴和纵轴上的数字及单位。
      • 考查图表数据及术语是否表达到位。
    • 柱状图题型要求考生通过宽度相等的柱形的高度或长度差别来判断事物的动态发展趋势，故考生应密切关注坐标线上的刻度单位及图表旁边的提示说明与文字。
      • 纵横推论：纵坐标一般为数据，横坐标一般是年代，抓住首尾两端数据，中间年代点到为止。
    • 饼状图题型旨在要求考生准确理解并阐述一个被分割成大小不等切块的圆形图所传达的信息。考生应清楚掌握部分与整体、部分与部分之间的相互关系，这种关系通常是以百分比的数字形式给出的。
      • 所有图表中最简单的一种，只需掌握两个词组：由……组成：consist of、be made up of、be composed of；占多大比例：account for、make/take up

根据最新考研大纲，图表作文第一段一般是图表描述段，属于描述，不能发表太多议论，在最近几年中，第一段考的都是图表描述；第二段一般是原因列举段，分析图表数据的内在变化原因；第三段一般是归纳结论段，对图表所反映的问题发表自己的议论或看法。

图表类大作文分为三段式进行写作，基本框架如下（与图画类相似，主要结合范文理解）：

  1. Introduction：
    • 总体描述
    • 挖掘细节词
  2. Body：
    • 主题句
    • 论证
    • 小结：将论证进行总结，与本段主题句首尾呼应
  3. Conclusion：
    • 结论句
    • 包装结尾，展望未来
